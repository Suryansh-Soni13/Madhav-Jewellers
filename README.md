# Madhav Jewellers - Premium Boutique Website

This website is a **fully standalone, realistic digital storefront**. No server or installation is required—simply open the HTML files in your browser.

## 🚀 How to View

1.  **Open Local Files**: Simply double-click on `index.html` to open it in any modern web browser (Chrome, Edge, Safari, etc.).
2.  **Navigation**: Use the elite navigation bar to move between Home, About, Collection, and Contact pages.
3.  **Test Images**: Open `test-images.html` to verify all jewellery images are loading correctly.

## 🏛️ Features
-   **Realistic "Live" Rates**: The gold rates are synced across all pages using your browser's local storage.
-   **Premium Jewellery Images**: All images are high-quality, locally stored jewellery photographs featuring authentic gold necklaces, bangles, chokers, earrings, chains, and more.
-   **Bespoke Tailoring Studio**: A realistic price calculation engine for custom pieces with a built-in PDF invoice generator.
-   **No-Server Required**: Works perfectly offline or directly from your desktop.
-   **Local Image Assets**: All jewellery images are stored in the `images/` folder for fast loading and offline functionality.

## 🔐 Admin Control
To update the "live" gold rates for the entire site:
1.  Open `admin.html` in your browser.
2.  Adjust the **Gold Rate** or **Labour Charges**.
3.  Click "Apply Global Changes".
4.  The new rates will instantly reflect on the Home page ticker and the Tailoring Studio!

## 📂 Project Structure
-   `index.html`: The prestigious entrance.
-   `about.html`: The 2005 heritage story.
-   `products.html`: The jewelry archive.
-   `contact.html`: Boutique booking portal.
-   `product-detail.html`: Customization and PDF invoice engine.
-   `admin.html`: The secret rate-management panel (hidden from main navigation).
-   `test-images.html`: Image verification test page.
-   `index.css`: The "Ivory Silk & Gold" design system.
-   `images/`: Folder containing all jewellery product images.
